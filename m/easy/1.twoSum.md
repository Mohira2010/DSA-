# 1. Two Sum

- **Murakablik**: Easy
- **Tegishlik Mavzular**: Array, Hash Table

## Masala

- Butun sonlardan iborat [nums] massivi va [target] deb nomlangan bitta butun son berilgan.
nums ichidan 2 ta sonning yig‘indisi [target] ga teng bo‘ladigan sonlarning indekslarini qaytaring.

- Siz har bir kiruvchi ma’lumot uchun faqat bitta to‘g‘ri javob bor deb hisoblasangiz bo‘ladi va bir xil elementni ikki marta ishlatish mumkin emas.

- Javobni istalgan tartibda qaytarishingiz mumkin.

### Example

Input: `nums = [2, 7, 11, 15]`, `target = 9`,

Output: [0,1]

Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].


## Javob-1 (1805 ms, 18.3 MB)

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        for i in range(len(nums)):
            for j in range(i + 1, len(nums)):
                if nums[i] + nums[j] == target:
                    return i, j
```

## Javob-2 (0.894s ms ,52 MB)

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        # 'two pointer' faqatgina array sorted bo`lgandagina to`g`ri ishlaydi
        left , right = 0, len(nums)-1
        while left < right:
            x = nums[left] + nums[right]
            if x == target:
                return [left+1,right+1]
            elif x < target:
                left += 1
            else:
                right -= 1
```
